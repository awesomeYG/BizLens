# AI-Native BI 产品设计方案

## 产品定位

**一句话**: 中小企业的第一个 AI 数据分析师

**目标用户**: 10-500 人规模企业，无专业数据团队

**核心价值**: 
- 零学习成本 - 会聊天就会用
- 分钟级上线 - 连接数据即可用
- 按需付费 - 中小企业用得起

---

## 设计原则

```
┌────────────────────────────────────────────────────┐
│                                                    │
│   Less is More                                     │
│   少即是多                                          │
│                                                    │
│   • 一个核心界面 - 对话即分析                        │
│   • 两个基础操作 - 提问 + 查看                       │
│   • 三个关键场景 - 看数 + 做表 + 分享                │
│                                                    │
└────────────────────────────────────────────────────┘
```

### 设计参考

| 产品 | 借鉴点 | 简化点 |
|------|--------|--------|
| **Querio** | 代码透明、Notebook 工作流 | 去掉 Notebook，隐藏代码 |
| **ThoughtSpot** | 自然语言搜索、Liveboard | 去掉复杂建模、权限体系 |
| **Linear** | 极致简洁、键盘优先 | - |
| **Arc Browser** | AI 优先、渐进式披露 | - |

## 商业模式

### 定价策略 (适合中小企业)

| 版本 | 价格 | 包含内容 | 目标客户 |
|------|------|----------|----------|
| **免费版** | ¥0 | 1 个数据源，100 次查询/月，基础图表 | 个人/初创 |
| **团队版** | ¥299/月 | 5 个数据源，无限查询，5 个报表，协作 | 小团队 |
| **专业版** | ¥999/月 | 无限数据源，AI 洞察，API 访问，白标 | 中小企业 |
| **企业版** | 定制 | 私有部署，定制开发，SLA | 大型企业 |

### 获客策略

1. **产品驱动增长 (PLG)**
   - 免费版足够好用，自然传播
   - 内置"通过 XX 制作"水印

2. **内容营销**
   - 数据分析最佳实践博客
   - 行业报表模板库

3. **合作伙伴**
   - 云厂商市场 (阿里云、腾讯云)
   - SaaS 集成 (钉钉、企业微信)

## 风险与对策

| 风险 | 对策 |
|------|------|
| NL2SQL 准确率不高 | 人工校验 + 持续训练 |

## 总结

**产品 essence**:

```
不是 "更简单的 BI 工具"
而是 "会数据分析的 AI 助手"
```

**差异化**:

| 传统 BI | 我们 |
|--------|------|
| 拖拽建模 | 对话即分析 |
| 学习成本高 | 零学习成本 |
| 面向分析师 | 面向业务人员 |
| 昂贵 | 中小企业用得起 |
| 复杂功能 | 核心功能极简 |

**设计 essence**:

```
ChatGPT 的交互体验
+
Tableau 的可视化能力
-
所有复杂的企业级功能
=
中小企业的 AI 数据分析师
```
