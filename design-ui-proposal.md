# UI 设计方案 - 本土化 AI 数据分析平台

## 一、竞品分析总结

### Querio.ai 设计特点
- **极简代码优先风格**: 以 Python 代码片段作为核心视觉元素
- **暗色主题**: 深色背景 + 亮色代码高亮
- **开发者友好**: 强调 Notebook、SQL、Python 等工作流
- **透明化 AI**: 每个 AI 查询都显示为可编辑的代码
- **模块化布局**: Explore / Notebook / Context / Boards 四大核心模块

### ThoughtSpot 设计特点
- **企业级专业感**: 蓝色主色调 (#00d2ff), 大量留白
- **Agent 矩阵展示**: Spotter 系列 Agent 的卡片式布局
- **客户证言驱动**: 大量 Logo 墙和客户引用
- **分层信息架构**: 按角色 (业务/数据/产品)、行业、规模分类
- **动效丰富**: 波浪、渐变、悬停效果

## 二、本土化设计理念

### 核心设计原则

| 原则 | 说明 | 实现方式 |
|------|------|----------|
| **中西合璧** | 国际范 + 本土亲和 | 现代极简布局 + 中式色彩/图形元素 |
| **信任优先** | 中国企业更重视安全可信 | 安全合规标识前置、客户案例突出 |
| **效率导向** | 快节奏决策文化 | 关键信息 3 秒可达、减少层级 |
| **人情味** | 不同于西方的冷科技感 | 温暖配色、拟人化 Agent 形象 |

### 色彩体系

```
主色调:
- 科技蓝：#2563EB (信任、专业)
- 活力橙：#F97316 (创新、活力 - 本土化元素)

辅助色:
- 成功绿：#10B981
- 警告黄：#F59E0B
- 危险红：#EF4444

中性色:
- 深空黑：#0F172A (主背景)
- 石板灰：#64748B (次要文字)
- 云雾白：#F8FAFC (卡片背景)

渐变:
- 主渐变：linear-gradient(135deg, #2563EB 0%, #7C3AED 100%)
- 本土渐变：linear-gradient(135deg, #F97316 0%, #EF4444 100%)
```

### 字体方案

```
中文优先:
- 标题：Noto Sans SC / 思源黑体 Bold
- 正文：Noto Sans SC / 思源黑体 Regular
- 代码：JetBrains Mono / Fira Code

英文字体 (fallback):
- Inter / system-ui
```

1. **Hero Section 动效**

   - CTA 按钮：悬停时发光效果 + 轻微放大

2. **中式元素融入**
   - 使用"祥云"图案作为 Section 分隔符 (低透明度)
   - 红色点缀用于重要按钮和徽标
   - 字体选择思源黑体，更符合中文阅读习惯

3. **本土化文案**
   - 避免直译英文，使用地道中文表达
   - 强调"降本增效"、"数据驱动"、"国产化"等关键词
   - 使用具体数字增强说服力

#### 关键特性

1. **智能图表推荐**
   - 根据数据特征自动选择最佳图表类型
   - 支持一键切换不同图表

2. **SQL 透明化**
   - 始终显示生成的 SQL
   - 支持手动编辑 SQL
   - AI 解释复杂 SQL 逻辑

3. **快速迭代**
   - 拖拽字段快速调整
   - 点击图例快速筛选
   - 双击标题快速编辑

## 六、设计系统组件

### 基础组件

- **按钮**: 主按钮 (蓝色渐变)、次按钮 (白色描边)、文字按钮
- **输入框**: 带图标、带验证状态、带自动补全
- **卡片**: 基础卡片、可悬停卡片、带操作卡片
- **表格**: 基础表格、可排序表格、可筛选表格
- **图表**: 统一配色、统一交互、统一图例

### 业务组件

- **数据卡片**: 指标卡、趋势卡、预警卡
- **Agent 对话框**: 对话气泡、建议问题、加载状态
- **文件上传**: 拖拽区域、进度条、预览
- **权限设置**: 角色选择器、资源树、操作矩阵

## 七、图标系统

### 风格定义

- **线性图标**: 2px 描边，圆角端点
- **面性图标**: 用于重要操作和选中状态
- **配色**: 单色 (当前文字色) / 彩色 (功能色)

### 核心图标

```
📊 数据相关：仪表盘、报表、图表、数据源
🤖 AI 相关：Agent、智能、自动、学习
🔧 操作相关：新建、编辑、删除、分享、下载
👤 用户相关：个人、团队、权限、角色
🔔 通知相关：消息、提醒、预警、错误
```

## 八、无障碍设计

### 对比度要求

- 正文文字：对比度 ≥ 4.5:1
- 大标题：对比度 ≥ 3:1
- 图标：对比度 ≥ 3:1

### 键盘导航

- Tab 键顺序合理
- 焦点状态明显 (蓝色外环)
- 支持快捷键

### 屏幕阅读器

- 所有图标有 aria-label
- 图表有文字描述
- 表单有正确 label
