# DataMind - AI 原生数据分析平台

> **让每个企业都拥有 AI 数据分析师**

DataMind 是一个极简的 AI-Native BI 平台，通过自然语言对话分析企业数据，零学习成本，分钟级上线。

---

## ✨ 核心特性

### 🤖 Chat-First 设计
- **像聊天一样简单** - 会提问就会分析
- **自动可视化** - AI 自动生成最佳图表
- **零学习成本** - 业务人员直接使用

### 📊 简化的核心功能
- ✅ 自然语言问答
- ✅ 自动图表生成  
- ✅ 智能报表 (Liveboard)
- ✅ 一键分享
- ❌ 复杂权限 / 复杂建模 / 自定义 SQL

### 🎨 高端极简 UI
- 暗色主题优先
- 大量留白 + 精致排版
- 参考 Linear / Vercel / Arc 的美学

## 🚀 快速开始

### 环境要求
- Node.js 18+
- Go 1.21+
- SQLite (开发环境) / PostgreSQL (生产环境)

### 1. 启动后端

```bash
cd backend
go run ./cmd/main.go
```

后端将在 `http://localhost:3001` 启动

### 2. 启动前端

```bash
cd frontend
npm install
npm run dev
```

前端将在 `http://localhost:3000` 启动

### 3. 访问应用

打开浏览器访问：http://localhost:3000

## 🏗️ 技术架构

### 前端
- **框架**: Next.js 15 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **设计**: 暗色主题 + 玻璃态卡片

### 后端
- **语言**: Go 1.21+
- **框架**: net/http (标准库)
- **ORM**: GORM
- **数据库**: SQLite / PostgreSQL

### AI
- **模型**: OpenAI GPT-4 / GPT-4o-mini
- **能力**: NL2SQL、自动洞察、智能图表推荐

## 💡 使用示例

### 1. 快速开始
```
1. 访问首页
2. 点击"快速测试"(使用演示数据)
3. 进入对话界面
```

### 2. 开始分析
```
示例问题:
- "上周销售额是多少？环比增长如何？"
- "本月营收趋势分析"
- "哪个产品卖得最好？"
- "预测下个月的销售额"
```

### 3. 连接数据源
```
1. 点击"连接数据源"
2. 选择数据库类型 (MySQL/PostgreSQL 等)
3. 输入连接信息
4. 测试并连接
```

### 4. 创建报表
```
1. 在对话中让 AI 分析
2. 点击"我的报表"
3. 查看和管理已创建的报表
```

## 🎨 设计原则

### Less is More
- 一个核心界面 - 对话即分析
- 两个基础操作 - 提问 + 查看
- 三个关键场景 - 看数 + 做表 + 分享

### 目标用户
| 维度 | 定位 |
|------|------|
| 公司规模 | 10-500 人 |
| 数据团队 | 无/1-2 人 |
| 使用场景 | 日常看数 + 周报月报 |
| 付费能力 | ¥300-1000/月 |
